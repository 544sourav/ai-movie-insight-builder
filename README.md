# AI Movie Insight Builder

## Overview
AI Movie Insight Builder is a full-stack application that allows users to enter an IMDb movie ID and instantly view movie details along with AI-generated audience sentiment insights.

## Features
- Fetch movie metadata using OMDb API
- Display movie poster, plot, cast, ratings, and metadata
- AI-powered audience sentiment analysis using Gemini 2.5 Flash
- Clean responsive UI
- Error handling and input validation
- Component-based architecture
- Basic unit testing with Jest

## Tech Stack

Frontend
- Next.js 16 (App Router)
- TypeScript
- TailwindCSS

Backend
- Next.js API Routes

APIs
- OMDb API
- Google Gemini AI

Testing
- Jest

Deployment
